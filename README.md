# Meu Site Estático

Projeto exemplo gerado automaticamente — site estático simples com HTML, CSS e JavaScript.

Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/geremiasedocha-beep/Meu-site.git
   cd Meu-site
   ```
2. Abra o arquivo `index.html` no navegador (duplo clique ou use um servidor local).

Sugestões de deploy

- GitHub Pages: vá nas configurações do repositório → Pages e publique a branch `main`.
- Netlify / Vercel: faça upload do repositório ou conecte pelo Git.

Editar

- Abra no VS Code: `code .`
- Edite os arquivos em `css/` e `js/` ou altere `index.html`.
